# Patent Compass

AI-powered Patent Research and Analysis Platform.

## Project info

A comprehensive tool for patent research, transcription, and structured briefing.

## Development

The only requirement is having Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server.
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- TanStack Query
- React Router
